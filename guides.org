** Switch
   	- D-link web smart switch 1224TP
   	- Configured at IP '10.90.90.90' subnet '255.255.255.0'
 	- Running jumboframes, thus MTU on everything should be 9000
*** instcructions for access
    	1. Connect a computer physically to it, any port works.
   	2. Set the computers static IP address to '10.90.90.91' (on the interface used to connect to the switch)
  	3. Set subnet mask to '255.255.255.0'
  	4. In a browser enter 10.90.90.90
    	5. Creds:	
    		- User: admin
 		- Password: In pass on laptop

** Nodes
	- Each node is booteed through iPXE from the controller, they run Rocky Linxu 9 as well

	- Each runs slurmd
	- Each should have a 'mpiuser' with Uid '1100'
